# Sistema de Gestión POS y BackOffice

Aplicación web desarrollada en ASP.NET que administra la creación de autorizadores para las cajas.

## Compatibilidad
* .NET Framework 4.6.2

## Dependencias de paquetes Nuget
* Autofac = 5.2.0
* Autofac.Mvc5 = 5.0.0
* AutoMapper 10.1.1
* MediatR = 9.0.0
